## Description

A [Spring Boot](https://github.com/spring-projects/spring-boot)  project setup with MongoDB integration.

## Installation

```bash
# install dependencies
$ ./gradlew build
```

## Running the app

```bash
# build
$ ./gradlew bootRun

# production
$ java -jar build/libs/<application-name>.jar
```

## Test

```bash
# unit tests
$ ./gradlew test
```
